# Ryu
Ryu 在 SDN 中負責的是管理部分，也就是 Controller 的角色。Ryu 是透過 Python 進行編寫，可結合實體設備或 Mininet 運行。Ryu 可以依需求建立封包處理方式，並執行封包配送、封包監控等功能。
