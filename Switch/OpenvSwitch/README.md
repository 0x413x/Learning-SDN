# Open vSwitch

Open vSwitch 是產品性質的軟體，它的授權標準為 Apache 2.0。設計來讓網路可透過程式自動化擴展，且可適用多種網路協定（e.g. NetFlow, sFlow, IPFIX, RSPAN, CLI, LACP, 802.1ag）。
