# SDN-tutorial
在此，將會介紹 SDN 的相關知識，並說明使用到的套件及操作方式。
### 學習項目
* 安裝 Mininet
* 使用 Mininet，建立虛擬網路環境
* 安裝 Ryu
* Mininet 連接 Ryu
* 第一個 Ryu 程式（實現一個基本的 Switch）
* 第二個 Ryu 程式（Traffic Monitor）
* Pica8 P-3297 實際操作-連結與模式設定
* Pica8 P-3297 實際操作-OVS基本設定
* Pica8 P-3297 實際操作-連結Controller


### 使用到的套件
* Mininet（虛擬網路環境）
* Ryu（SDN Controller）
* GtkTerm（連接週邊硬體）

### 硬體設備
* Pica8 P-3297
