# SDN-tutorial
在此，將會介紹SDN的相關知識，並說明使用到的套件及操作方式。
### 學習項目
* 安裝Mininet
* 使用Mininet，建立虛擬網路環境
* 安裝Ryu
* Mininet連接Ryu
* 第一個Ryu程式（實現一個基本的Switch）
* Pica8 P-3297實際操作-連結與模式設定
* Pica8 P-3297實際操作-OVS基本設定
* Pica8 P-3297實際操作-連結Controller


### 使用到的套件
* Mininet（虛擬網路環境）
* Ryu（SDN Controller）
* GtkTerm（連接週邊硬體）

### 硬體設備
* Pica8 P-3297
